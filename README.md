# Heart Disease Prediction System

## Overview

This project is a machine learning-based web application that predicts the likelihood of heart disease using patient health information. The application is developed using Python, Scikit-learn, and Streamlit.

## Features

* Predicts heart disease based on medical parameters
* Simple and interactive web interface
* Machine learning model trained using the K-Nearest Neighbors (KNN) algorithm
* Real-time prediction through Streamlit

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Joblib

## Dataset

The project uses a heart disease dataset containing various health-related attributes such as age, blood pressure, cholesterol level, chest pain type, and other medical factors.

## How to Run

1. Clone the repository

git clone https://github.com/SachinDevji/heart-disease-prediction.git

2. Move to the project folder

cd heart-disease-prediction

3. Install the required packages

pip install -r requirements.txt

4. Run the application

streamlit run app.py
